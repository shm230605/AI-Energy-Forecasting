# ⚡ AI Energy Consumption Forecasting System

## 📌 Project Overview
This project predicts electricity consumption using Machine Learning (Random Forest / MLP).

## 📊 Features
- Energy usage forecasting
- Time-series analysis
- Flask API for predictions
- Visualization of results

## 🧠 Tech Stack
Python, Pandas, Scikit-learn, Flask, Matplotlib

## 🚀 How to Run

```bash
pip install -r requirements.txt
python main.py
python app.py
```
